# Quiz-Generator
Launched an AI-Powered Etea Quiz Generator!  Built a complete system using Python, Streamlit, Groq LLM, and Pydantic that dynamically generates MCQs and Fill-in-the-Blank questions, evaluates answers, and download results . It generates random MCQs and Fill in the Blanks evaluates answers, and provides instant scoring using Groq LLM + Streamlit.
